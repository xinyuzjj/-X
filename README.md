# 🦁 My Trading System (我的交易系统)

> **核心理念**：基于价格行为 (Price Action) 与 聪明钱概念 (SMC)，通过识别市场结构、关键流动性区域与斐波那契数列，捕捉高盈亏比的交易机会。

## 1. 读懂 K 线 (Candlestick Patterns)

K线是所有分析的基础。我主要关注以下几种反转形态：
* **吞噬结构 (Engulfing)**：反转的最强信号。
* **黄昏之星 / 希望之星**：顶部或底部的明确信号。
* **🔥 OB (Order Block) 核心定义**：
    > **我的 OB 定义**：当出现 **吞噬形态** 并且随后紧跟 **三连涨（或三连跌）** 时，这不仅仅是反转，这是机构建仓的痕迹。

## 2. 关键时间与开盘价 (Time & Price)

时间决定了波动率，开盘价决定了日内多空分界。

### ⏰ 关键时区 (UTC+8)
* **亚洲盘 (08:00 - 14:00)**：通常作为日内震荡区间。
* **伦敦盘 (15:00 - 20:00)**：诱多诱空的高发期。
* **🇺🇸 纽约盘 (21:00 - 24:00)**：**核心交易时间**。
    * *注：纽约开盘时间 21:30，新闻数据发布最频繁，波动最大。*

### 🔑 关键开盘价
我关注以下开盘价作为强支撑/阻力：
1.  **周开 (Weekly Open)**：力量最强，周一开盘的大阳/大阴往往奠定本周基调。
2.  **日开 (Daily Open)**：日内交易的多空分水岭。
3.  **4H 开 (4H Open)**：配合 15分钟级别寻找入场点。

---

## 3. 市场结构 (Market Structure)

交易的核心是**跟随趋势**或**捕捉反转**。

* **结构延续 (BOS)**：在强趋势中，价格不断突破前高/前低，且回调很小（目标是猎杀前方流动性）。
* **结构改变 (CHoCH)**：
    * **强结构**：震荡区间中的假突破（Fakeout）或接跌破，这代表趋势非常强劲。
    * **操作逻辑**：无论是延续还是改变，**必须等待回调**，不要追单。

---

## 4. 关键区间 (Key Zones)

价格永远在这些区间之间移动。

1.  **OB 区间 (Order Block)**：
    * 以前述“吞噬+三连”的起始吞噬K线作为 OB 区域。
    * **高胜率技巧**：当价格后续首次回测此 OB 区间时，是最佳进场点。
2.  **FVG (Fair Value Gap)**：
    * 强趋势留下的价格真空区。
    * **筛选标准**：只取那些位于整段趋势 **50% 以上** 位置的 FVG，其他的忽略。
3.  **流动性 (Liquidity)**：
    * 庄家插针的原因就是为了获取买卖双方的流动性（止损盘）。价格总是向着流动性移动。

---

## 7. 斐波那契回测 (Fibonacci Retracement)

斐波那契是我判断回调深度的标尺。

| 数值 | 含义 | 用法 |
| :--- | :--- | :--- |
| **0.382** | 强趋势支撑 | 如果回调跌不破 0.382，说明上涨趋势极强。 |
| **0.5** | 均衡点 | 突破 0.5 是趋势延续的参考标准。 |
| **0.618 - 0.79** | **OTE (最佳入场位)** | **核心开单区**。深度回测位，通常结合结构改变使用。 |
| **1.113** | 假突破 | 用于判断猎杀止损后的反转 (Shark Fin)。 |

## 8. 成交量 (Volume)

* **爆量**：当价格到达关键位置时出现异常大的成交量。
* **判断**：这通常意味着**趋势反转**（恐慌抛售/高潮买入）或者**强力突破**（机构进场）。

---

## ⚡ 执行系统 (Execution)

### 🎯 进场与出场 (Entry & Exit)
**我的进出场逻辑基于“共振”：**
* **进场**：价格回踩 **OB** + 处于 **OTE (0.618-0.79)** + 扫掉 **流动性**。
* **出场**：也就是反向的 OB、流动性池或斐波那契扩展目标位。

### 🛡️ 止损 (Stop Loss)
* **规则**：严格设置在关键区间（如 OB 或 FVG）的**下方/上方**。
* *宁可被打止损，也不扛单。*
